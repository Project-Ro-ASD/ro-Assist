**🇹🇷 Türkçe:**
ro-Assist, Fedora Linux için geliştirilmiş, C++ ve Qt6 tabanlı modern bir sistem yönetim asistanıdır. `dnf` paket yöneticisi ile doğrudan (backend) iletişim kurarak sistem güncellemelerini güvenli ve gerçek zamanlı bir ilerleme çubuğu ile gerçekleştirir.

**Özellikler:**
* **Gerçek Zamanlı Takip:** Terminal çıktılarını (RegEx) okuyarak yüzde bazlı ilerleme gösterimi.
* **Güvenli Sudo:** Şifreleme ve hata kontrolü mekanizmaları.
* **Modern Arayüz:** QSS ile özelleştirilmiş animasyonlu butonlar ve modern tasarım.

---

**🇬🇧 English:**
ro-Assist is a modern system management assistant for Fedora Linux built with C++ and Qt6. It communicates directly with the `dnf` package manager to perform system updates securely with a real-time progress bar.

**Developer:** Ebubekir Bulut  
**Year:** 2026
